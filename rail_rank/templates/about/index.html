About

This website

This is a web application that monitors the performance of multiple train operators in the United Kingdom.

It is a work in progress, and is currently in the early stages of development. Its code is available on GitHub if you are into that thing.

The data

This website uses data from the public National Rail API. Performance data is measure with respect to the day, but updated in real-time. This website periodically queries the API for the latest performance data, and stores it in a database. The data is then used to generate the content on this website.
